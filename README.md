# Formula Team Manager 1970

Manageriale Formula 1 ambientato negli anni '70.

## Funzioni V1

- Registrazione
- Login
- Dashboard
- Simulazione gara
- Budget iniziale
- Salvataggio database MySQL

## Installazione

1. Crea database MySQL.
2. Importa database.sql.
3. Configura config/database.php.
4. Carica i file sul server.
